Fitness Tracker API (Golang) — backend-сервис для управления задачами и тренировками с автоматическим расчётом дат выполнения; реализована архитектура с разделением на слои (handlers, business logic, database), разработано REST API (CRUD-операции), настроена работа с SQLite через database/sql, реализован алгоритм NextDate с учётом граничных случаев, добавлены валидация и обработка ошибок, покрытие ключевой логики unit-тестами; результат — готовый к расширению и интеграции backend-сервис; стек: Go (Golang), SQLite, database/sql, HTTP, REST API, JSON, testing; навыки: backend-разработка, проектирование архитектуры, работа с БД, разработка API, тестирование.
